## 🚀 快速部署

### 方案1：Vercel（最简单，推荐）

**步骤：**
1. 访问 [https://vercel.com](https://vercel.com)
2. 注册/登录（支持GitHub账号）
3. 点击 "Add New" → "Project"
4. 拖拽本文件夹到上传区域
5. 点击 "Deploy"

**完成！** 获得地址：`https://h5-click-game.vercel.app`

### 方案2：GitHub Pages（免费）

**步骤：**
1. 创建GitHub仓库：`h5-click-game`
2. 上传本文件夹所有文件
3. 仓库设置 → Pages → 选择 main 分支
4. 保存

**完成！** 获得地址：`https://你的用户名.github.io/h5-click-game`

### 方案3：Netlify（拖拽上传）

**步骤：**
1. 访问 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 拖拽本文件夹到页面
3. 等待部署完成

**完成！** 获得地址：`https://随机名称.netlify.app`

⚠️ 常见问题解决

如果"拒绝连接"：

  1. 检查Docker端口映射：容器是否暴露了8080端口？
  2. 检查防火墙：主机防火墙是否允许8080端口？
  3. 检查网络模式：容器是否使用host网络？

快速诊断命令：

# 在宿主机执行
curl http://localhost:8080/  # 如果失败，端口未映射
docker ps                    # 查看端口映射
docker port <容器名>         # 查看具体映射
